Proyecto de Arduino del curso Arquitectura de Computadores del Segundo Semestre 2016 del Tecnológico de Costa Rica.

Arduino
UNO

El Arduino Uno es una plataforma computacional física que puede ser utilizado para desarrollar objetos interactivos.

Descripción

Básicamente lo que se quiere lograr con el proyecto es desarrollar un vehículo a control remoto, impulsado por Arduino.

¿Cómo lo haremos?

El Arduino será alimentado por una batería solar, este manipulará un motor con el que avanzará y un timón para dar dirección. Los materiales a utilizar son: un Arduino UNO, el control remoto que manejará la dirección del vehículo, un motor que será controlado por el circuito del vehículo y dictará la velocidad del auto. Llantas para darle la movilidad y una coraza que protege el circuito.

División del trabajo

Pasos a seguir para realizar el proyecto: 

  • Conseguir todos los materiales necesarios.
  • Comenzar la documentación del proyecto.
  • Diseñar la estructura del vehículo.
  • Comprobar que las partes funcionen.
  • Armar el vehículo.
  • Programar el comportamiento Arduino en ensamblador.
  • Conectar el ensamblador con el vehículo.
  • Crear el Prototipo.
  • Llevar a cabo pruebas.
  • Hacer las correcciones necesarias.
  • Crear la versión final.
  • Se revisa y completa la documentación 
  • Se crea la página web de cómo hacer, y cómo utilizar el proyecto.

El Arduino tendrá..

  • Motores
  • Bluetooth
  • Baterías
  • Manejado mediante un Smartphone
